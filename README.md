# dwebssh

通过web方式连接至服务器的shell终端

#### 主要功能

- 连接至服务器的终端shell
- 管理服务器列表
- 管理用户列表，简单的用户权限管理

#### 所需组件

- python 3.6
- django 2.1.3（python模块）
- paramiko 2.4.2（python模块）
- dwebsocket 0.5.10（python模块）
- pymysql 0.9.2（python模块）
- bootstrap 3.3.6
- jquery 1.9.1
- fontawesome 4.7.0
